# Rails Blog


## Installation

 clone the repo
```
https://github.com/Yassir4/rails-blog.git
```

```
$ bundle install 
```

next 

```
$ rails db:migrate
$ rails db:seed
```

run the test suite to verify that everything is working correctly:

```
$ rspec spec
```

If the test suite passes, you'll be ready to run the app in a local server:

```
$ rails server
```
